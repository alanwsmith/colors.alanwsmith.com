Content For Launch

TL;DR

Play around with the Themes, Background, and Colors controls in the sidebar to change colors.

Hit Export above when you've got a palette you like. It'll give you a stylesheet with your colors.

There is no step 3.

The Basics

Color Themes

There are four themes: Light, High-Contrast Light, Dark, High-Constast Darkpitch for more themes. The buttons in the Themes box change between themguidelines.

The Background Color

The three sliders in the Background box are l, c, and h (aka lightness, chroma, and hue). Changing them changes the background color. Fiddle with them to get an idea of how they work.

You can use the Isolate checkbox to hide content. It makes it easier to focus on the color.

The Colors

The picker has five colors: content, link, title, heading, and accentmore soon. Change between them with the colored tabs in the Colors box.

Clicking/tapping a colored set from the grid changes the selected color to match it. The c slider below the set grid changes the chroma of the colormore later. Basically, its intensity.

The Isolate checkbox in the Colors box hides everything that's not using the selected color.

Picking

That's it. There's not a lot too it. That's kinda the point.

I make lots of websites. Picking colors for them was always a pain with existing toolsnote about other tools. A friction that slowed me down with no payoff. I built this picker to address that. It's designed to generate a nice set of colors with as little effort as possible.

I hope you enjoy it,

-a
