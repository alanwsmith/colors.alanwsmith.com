TL;DR

Play around with the Themes, Background, and Colors controls in the sidebar to change colors.

Hit Export above when you've got a palette you like. It'll give you a stylesheet with your colors.

There is no step 3.

The Basics

Color Themes

There are four themes: Light, High-Contrast Light, Dark, High-Constast Darkpitch for more themes. The buttons in the Themes box change between themguidelines.

The Background Color

The three sliders in the Background box are l, c, and h (aka lightness, chroma, and hue). Changing them changes the background color. Fiddle with them to get an idea of how they work.

You can use the Isolate checkbox to hide content. It makes it easier to focus on the color.

The Colors

The picker has five colors: content, link, title, heading, and accentmore soon. Change between them with the colored tabs in the Colors box.

Clicking/tapping a colored set from the grid changes the selected color to match it. The c slider below the set grid changes the chroma of the colormore later. Basically, its intensity.

The Isolate checkbox in the Colors box hides everything that's not using the selected color.

Picking

That's it. There's not a lot too it. That's kinda the point.

I make lots of websites. Picking colors for them was always a pain with existing toolsnote about other tools. A friction that slowed me down with no payoff. I built this picker to address that. It's designed to generate a nice set of colors with as little effort as possible.

I hope you enjoy it,

-a

Set Color Mechanics

The picker users OKLCH2. The L, C, and H being our lightness, chroma, and hue. Each of the set colors is based off the the h/hue value from the Background.

Behind the scenes, the Background h/hue value is between 0 and 360. The first from of the set colors adds 45 to it. Each subsequent row adds another 45flips.

  1. Move the "l", "c", and "h" sliders in the "Background" box around to find a background color you like
  2. Click/Tap some of the "set" buttons in the "Colors" box to choose a text color
  3. Move the "c" slider around in the "Colors" box around to change the intensity of the color
  4. Check out the other features below to learn more about how things work
  5. Click/Tap the "Export Colors" tab above to get your colors

Previewing

Click/Tap any of the content in the main part of the page to hide the color picker controls. This is designed to make it easier to evaluate palettes without the other options influencing the way things look.

Other Colors

Click the single letter tabs at the top of the "Colors" box to change which text the "set" colors effect (e.g. the main content, the title, links, etc.).

Isolating Colors

The "Isolate" button at the bottom of the "Color" box hides all the content except for what the selected color applies to. I added it to make it easier to find links and accents which might only appear on a page a few times.

Isolating The Background

The "Background" box also has an "Isolate" checkbox. Ticking it hides all the content so you can pick the background without existing text colors influencing it.
