    made by alan w smith

    TL;DR

    Play around with the controls in the Modes, Background, and Colors boxes to change colors.

    Use the Export tab to get a stylesheet when you've got a palette you like.

    The Basics

    Color Modes

    There are four color modes to choose from: Light, High-Contrast Light, Dark, High-Constast Dark. The buttons at the top of the sidebar change between them.

    The Background Color

    The three sliders in the Background box are l, c, and h (which stand for lightness, chroma, and hue, respectively). Changing them changes the background color.

    You can use the Isolate checkbox in the Background to hide content and make it easier to focus on the color.

    The Colors

    The picker provides five colors by default*: content, link, title, heading, and accent.

    1. Move the "l", "c", and "h" sliders in the "Background" box around to find a background color you like
    2. Click/Tap some of the "set" buttons in the "Colors" box to choose a text color
    3. Move the "c" slider around in the "Colors" box around to change the intensity of the color
    4. Check out the other features below to learn more about how things work
    5. Click/Tap the "Export Colors" tab above to get your colors

    Previewing

    Click/Tap any of the content in the main part of the page to hide the color picker controls. This is designed to make it easier to evaluate palettes without the other options influencing the way things look.

    Other Colors

    Click the single letter tabs at the top of the "Colors" box to change which text the "set" colors effect (e.g. the main content, the title, links, etc.).

    Isolating Colors

    The "Isolate" button at the bottom of the "Color" box hides all the content except for what the selected color applies to. I added it to make it easier to find links and accents which might only appear on a page a few times.

    Isolating The Background

    The "Background" box also has an "Isolate" checkbox. Ticking it hides all the content so you can pick the background without existing text colors influencing it.

    Light/Dark Modes

    The buttons above the "Background" box choose between on of four different color modes: "Light" "High-Contrast Light", "Dark", and "High-Contrast Dark". Each one is independent, but operates the same way.
